# Spring Cloud OpenFeign

## Spring Cloud CircuitBreaker Support

需要加入 `Spring Cloud CircuitBreaker` 依赖

并且，加入配置：`feign.circuitbreaker.enabled=true`

```yaml
feign:
  circuitbreaker:
    enabled: true
```
